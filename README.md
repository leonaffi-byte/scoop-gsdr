# scoop-gsdr

Scoop bucket for [GSDR](https://github.com/leonaffi-byte/gsdr) — Autonomous spec-driven development for Claude Code.

## Install

```powershell
scoop bucket add gsdr https://github.com/leonaffi-byte/scoop-gsdr
scoop install gsdr
```

## Update

```powershell
scoop update gsdr
```

## Uninstall

```powershell
scoop uninstall gsdr
scoop bucket rm gsdr
```
